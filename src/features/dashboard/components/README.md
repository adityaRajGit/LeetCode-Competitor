# Dashboard Components

This directory contains all React components for the competitive coding dashboard feature.

## Component Structure

Each component follows this structure:
- `ComponentName/`
  - `index.ts` - Barrel export
  - `ComponentName.tsx` - Main component
  - `ComponentName.module.css` - Component styles
  - `ComponentName.test.tsx` - Component tests

## Components

- **DashboardLayout**: Main container for the dashboard
- **HeroSection**: Personal stats and 1v1 comparison
- **Leaderboard**: Ranked friend list
- **ActivityFeed**: Recent friend activities
- **PointsSystemCard**: Scoring formula explanation
- **PerformanceCharts**: Visualization components
- **BadgeDisplay**: Achievement badges
- **AddFriendModal**: Friend addition flow
- **TopNavigation**: Top navigation bar
- **UsernameSetup**: First-time user onboarding
