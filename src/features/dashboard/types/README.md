# Dashboard Types

TypeScript type definitions for the dashboard feature.

## Type Definitions

- `LeetCodeUser` - User data model
- `Friend` - Friend with comparison metadata
- `ActivityEvent` - Activity feed event
- `Badge` - Badge definition and criteria
- `SubmissionCalendar` - Daily submission counts
- `PointsBreakdown` - Points calculation breakdown
- `LeaderboardEntry` - Leaderboard row data
- `Theme` - Theme type ('light' | 'dark')
