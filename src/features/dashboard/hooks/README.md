# Dashboard Hooks

Custom React hooks for the dashboard feature.

## Available Hooks

- `useUserStats` - Fetch and manage single user LeetCode statistics
- `useFriendsList` - Manage friend list in session storage
- `useLeaderboard` - Calculate rankings and points for leaderboard
- `useActivityFeed` - Track and generate activity events
- `useStreakCalculator` - Calculate streaks from submission calendar
- `usePerformanceData` - Process submission data for charts
