# Shared Directory

Contains reusable code shared across features.

## Structure

- **components/**: Reusable UI components (Button, Card, Modal, etc.)
- **hooks/**: Custom React hooks
- **utils/**: Utility functions and helpers
- **types/**: Shared TypeScript types and interfaces

## Guidelines

- Components here should be generic and reusable
- Avoid feature-specific logic
- Document props and usage with JSDoc comments
